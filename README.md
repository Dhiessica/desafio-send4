# FrontSend4-Teste

Desafio:
Escreva uma aplicação usando React e as bibliotecas da sua escolha, para construir a
seguinte cena: Um prédio com 12 janelas. Ao clicar nas janelas o usuário liga e desliga
as luzes das janelas. A cena também contém um controle com um botão que permite e
ligar e desligar todas as janelas. Use a Sunset Sunrise API
(https://sunrise-sunset.org/api) para mudar automaticamente a cena dependendo se
está noite ou dia no seu local. Ou seja, utilize API do browser de geolocalização.


Pontos concluídos:
